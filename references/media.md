# Educational Media of Note

Podcasts, articles, and talks worth keeping as reference.

---

## Context Engineering with Dex Horthy
The CEO of HumanLayer (who coined "context engineering") on how LLM context management has become a distinct engineering discipline. Covers the 12-Factor Agents framework, why LangChain-style abstractions get abandoned in production, loop engineering, harness engineering, and the evolving shape of AI-assisted software development.
[Context Engineering with Dex Horthy — The Pragmatic Engineer](https://newsletter.pragmaticengineer.com/p/context-engineering-with-dex-horthy)

## pytest Tips and Tricks for Better Testing
Brian Okken (author of *Python Testing with pytest*) on Talk Python To Me. Covers fixture patterns, parametrize, markers, plugins, and the deeper power hiding under pytest's simple surface. Good for both beginners and power users looking to level up their test suite.
[pytest Tips and Tricks — Talk Python To Me #407](https://talkpython.fm/episodes/show/407/pytest-tips-and-tricks-for-better-testing)

## Taming Flaky Tests
Gregory Kapphammer and Owen Perry on the research and practical strategies behind flaky tests — tests that pass and fail intermittently without code changes. Covers root causes (timing, ordering, external deps), detection strategies, and how to systematically eliminate them.
[Taming Flaky Tests — Talk Python To Me #429](https://talkpython.fm/episodes/show/429/taming-flaky-tests)
